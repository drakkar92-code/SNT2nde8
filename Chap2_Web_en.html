The Web

The web, or the World Wide Web, is a global system of interconnected pages and content accessible via the Internet, allowing users to browse, view, and share information through protocols like HTTP and web browsers.

  • The Web, or World Wide Web (WWW), is a public hypertext system operating on the Internet, allowing users to browse pages grouped into websites using a browser. The term "Web" refers to the hyperlinks that connect web pages, creating an image of a spider's web. The Web was invented in 1989-1990 at CERN by Tim Berners-Lee and Robert Cailliau. It has evolved significantly since then, becoming an essential tool for information retrieval, online shopping, and communication. Although the Web is often confused with the Internet, it is actually an application of the Internet, distinct from other services such as email or peer-to-peer file sharing.
